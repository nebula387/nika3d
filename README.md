# Nika 3D

Голосовой 3D-компаньон. Python обрабатывает речь и диалог, Godot
рендерит 3D-персонажа и анимации.

## Возможности (на 2026-07-22)

- **Распознавание речи** — Whisper, только английский (см. `CLAUDE.md` →
  "English-only pipeline" почему)
- **Диалог** — локальная LLM через LM Studio (`LLM_MODEL=auto` сама находит
  загруженную модель) либо облачный Groq как запасной вариант
- **Память сессии** — помнит последние несколько реплик разговора, пока
  работает процесс; фраза "forget everything" сбрасывает историю
- **Живой веб-поиск** — погода, курсы валют, новости, общий поиск
  (`web.py`) — отвечает реальными данными, а не выдумывает
- **Синтез речи** — на выбор: Piper (быстро, EN), XTTS-v2 (EN+RU,
  клонирование голоса, медленнее на CPU), ElevenLabs (облако)
- **Управление ПК голосом** — открыть сайт/программу, напечатать текст,
  поискать в Google (`actions.py`, выполняется локально в Python)

## ⚠️ Структура репозитория (нестандартная)

коммитом изменённые файлы копируются сюда, в `brain/`, вручную. Подробности
и почему так вышло — в `CLAUDE.md` → "ВАЖНО: структура репозитория".

## С чего начать

1. Прочитать `CLAUDE.md` — полная архитектура, история этапов, решения
   и почему они были приняты
2. Прочитать `docs/setup-godot.md` — установка Godot перед первым запуском
3. Открыть рабочую копию (`nika3d/brain/`, см. выше) как проект в Claude Code
4. Отправить текст из `docs/START_HERE.md` как первое сообщение (если
   начинаете с нуля; если продолжаете существующий проект — не нужно)

## Запуск brain/ (Python-часть)

1. Установить зависимости: `pip install -r brain/requirements.txt`
2. Запустить LM Studio, загрузить любую модель, нажать "Start Server"
   (порт 1234 по умолчанию)
3. Скопировать `brain/.env.example` → `brain/.env`, проверить/заполнить
   значения (пути к голосам Piper, ключи ElevenLabs/Groq если нужны)
4. `python brain/main.py`
5. Сказать "Nika" в микрофон, дождаться короткого сигнала, задать вопрос

## Структура

- `CLAUDE.md` — инструкции для Claude Code (контекст, архитектура, история
  решений, правила)
- `docs/protocol.md` — формат сообщений между Python и Godot
- `docs/setup-godot.md` — установка Godot, импорт модели
- `docs/setup-vrm.md` — замена персонажа на VRoid VRM
- `docs/START_HERE.md` — стартовый промпт для Claude Code (для запуска
  проекта с нуля)
- `brain/` — Python-часть (мозг) — см. предупреждение о структуре выше
- `godot_project/` — Godot-часть (тело), шаблоны скриптов (живой проект
  в `C:\Godot\nika-3d\`)
